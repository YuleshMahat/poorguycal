# PoorGuy Cal — Support site

Web support pages for **PoorGuy Cal** (React, Vite, TypeScript, Tailwind).

For bugs and issues, payment and license questions, or general contact, use the in-app support links or email **yuleshmahat2058@gmail.com**. You can also use the contact form at [www.yuleshmahat.com](https://www.yuleshmahat.com).

Enjoy PoorGuy Cal and happy tracking.

## Development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

The privacy policy source from the repo lives in `PrivacyPolicy.md`; the site’s **Privacy** route can mirror that content when published.
